# Rainier Dominguez — Portfolio

Simple dark-themed portfolio built for Rainier Dominguez (22, 4th-year IT student).

## Features

- Dark, minimal, responsive layout
- Scroll animations using IntersectionObserver
- Small page transition overlay for section navigation
- Sections: Hero, Skills, Projects (WeSign + Digital Drawings), **Internships (LANDBANK)**, Contact

## How to use / Customize

1. Edit `index.html` to update text, add images inside a new `assets/` folder.
2. Adjust colors and spacing in `styles.css` (CSS variables at the top).
3. Add more project cards under `#projects`.

## Deploy to GitHub + Vercel (recommended)

1. Create a new repo on GitHub (e.g. `rainier-portfolio`).
2. Push the files:

```bash
git init
git add .
git commit -m "Initial portfolio"
git remote add origin git@github.com:yourusername/rainier-portfolio.git
git branch -M main
git push -u origin main
```

3. Sign in to https://vercel.com/ and import the repo. Choose "Other" or static site — Vercel will auto-detect and deploy.

That's it — Vercel will redeploy on each push.
